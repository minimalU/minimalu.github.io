PRESCRIPTXN

Smart Cities Hackathon Finalist
- Python, Flask, mySQL, openAI API, JavaScript -
PRESCRIPTXN allows users to access patient-friendly prescription information that is AI-processed from FDA medication information. Users can easily check their medication information anytime, and can quickly access the clinic and receive their medication from the pharmacy using QR codes.

Manufacturing Simulator

Manufacturing Simulation Project
- C#, .NET WEB API, EntityFramework, SignalR -
- Microsoft SQL, WPF, Windows Forms -
The Manufacturing Simulation project is a System project composed of WEB API, Configuration tool, Simulator, Dashboards(Workstation, Replenishment, Kanban). The simulator updates the database according to the configuration for each workstation. The dashboards are updated with real-time data. Features are completed with stored procedures, triggers, views for SQL databases, Entity Framework for WEB API, and SignalR for real-time communication between applications.
